Chebfunnumerical computing with functions

  • x = chebfun('x');

% Define two functions
f = exp(-x).*tanh(4*sin(40*x)).*(2 + sin(7*x)) / 16;
g = (exp(sin(12*x) - x) + x - 2) / 12;

rr = roots(f - g);      % Compute their intersections

plot([f g]), hold on    % Plot the functions
plot(rr, f(rr), 'o')    % and their intersections

  • % Airy operator
L = chebop(@(x,u) 0.01*diff(u,2) - x.*u, [-5,5]);
L.bc = 'dirichlet';

% Solve
u = L \ 1;

plot(u)

Chebfun is an open-source MATLAB package for computing with functions to 15-digit accuracy. Most Chebfun commands are overloads of familiar MATLAB commands — for example sum(f) computes an integral, roots(f) finds zeros, and u = L\f solves a differential equation.
