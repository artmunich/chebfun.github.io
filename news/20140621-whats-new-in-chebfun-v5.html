What's new in Chebfun Version 5

04 June 2014

Today we're excited to announce the release of Chebfun Version 5!

Chebfun Version 5 represents a complete rewrite of the Chebfun code and a transition from a svn repository at Oxford to a public repository and issue tracker at GitHub. We are a fully open source project. We invite all our users to learn more and get involved.

The main purpose of the code rewrite has been greater modularity, clarity, and consistency, to make it easier for new developers to get involved and new variations to be explored and new features to be added. A completely new web site has also been created.

Some new features have also appeared in v5, including:

  • A single email address help@chebfun.org for all queries. More details.
  • First official release of Chebfun2. More details.
  • Big speedups of conv, polyfit, abs. More details.
  • New commands leg2cheb, cheb2leg. More details.
  • Algorithms based on Chebyshev points of the first kind. More details.
  • A periodic option for Fourier instead of Chebyshev. More details.
  • Ultraspherical spherical methods for ODEs. More details.
  • epslevel estimates of accuracy, still under development. More details.

The release of June 21, 2014 is a beta release, and Chebfun v5 is far from perfect.
