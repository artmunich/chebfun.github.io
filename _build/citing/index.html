Citing

To cite Chebfun in a publication, we recommend the following:

The Chebfun Team, Chebfun Version 5, 2014, http://www.chebfun.org/.

Below is a list of Chebfun-related papers written by members of the Chebfun Team. For convenience, a BibTeX file is available. If you use Chebfun in a book, paper, website, or other project, we'd love to hear about it!
